## Exercice 1 : Navigation Simple et Vérification de Titre


**Site**: example.com
**Compétences**: Démarrer Selenium, naviguer, vérifier le titre

**Instructions**:
1. Ouvrez un navigateur Chrome
2. Accédez à https://example.com
3. Vérifiez que le titre de la page est exactement "Example Domain"
4. Vérifiez que le contenu contient "Example Domain"
5. Fermez le navigateur

**Validation**:
- Le navigateur s'ouvre sans erreur
- La page se charge correctement
- Le titre est correct
- Le navigateur se ferme proprement
